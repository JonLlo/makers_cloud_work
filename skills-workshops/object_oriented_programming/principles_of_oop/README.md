# Object Oriented Programming 1

This workshop is the first in a series of two on OOP. In this workshop, we cover the 4 core principles of OOP

- Encapsulation
- Abstraction
- Inheritance
- Polymorphism

The second workshop covers the SOLID principles.

## Overview

It's easy to fall into the trap of assuming encapsulation, abstraction, inheritance and polymorphism to be completely independent from one another but that isn't the case - there are dependencies and there are overlaps. I like to think of them like this...

**Encapsulation –> abstraction –> inheritance –> polymorphisms**

Where the arrow means "allows" or "leads to".

This perspective might not make sense straight away, but by the end of this workshop everything should be a lot clearer.

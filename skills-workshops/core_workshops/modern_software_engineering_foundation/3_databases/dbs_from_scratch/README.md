## Building a Database-Backed Application from Scratch

During the course of this module, you'll use a starter project that we have built and you'll learn about building database-backed applications by extending it.

In this workshop, we'll build and extend a small database-backed application from scratch, with the aim of clarifying what all the different parts do and how they fit together.

### Intro

Your coach will demonstrate the first few steps of building the app from scratch. They won't initially build the application using the approach you've seen in the starter app – it will start simpler.

### Main

Your coach will share their code with you and you'll have several options in terms of what you do with it.

1. Use a tool like ChatGPT or Copilot to understand the code that already exists
2. Extend the existing codebase without any refactoring
3. Refactor the existing codebase to use the same approach as we've used in the starter codebase, then extend it

### Plenary

We'll wrap up the workshop by regrouping to reflect on the exercises, ask questions and share learnings.

# Welcome to the Python Katas!

In programming, a **kata** is a small, focused exercise designed to help you practice your "moves" until they become second nature—much like practising drills in martial arts.

These sessions are designed to help you sharpen your Python skills and practice collaborative problem-solving through **Pair Programming**.

### How it Works

1. **The Files:** You have been provided with a set of Python files (e.g., `kata1.py`, `kata2.py`).
2. **The Goal:** Inside each file, you’ll find a problem description in the comments. Your task is to complete the code so that it solves the problem correctly.
3. **The Execution:** You'll run your code directly from the command line to test your progress.
4. **Solutions:** There is rarely just one "right" way to solve these, so focus on writing clean, readable code. There will be time to compare solutions at the end.

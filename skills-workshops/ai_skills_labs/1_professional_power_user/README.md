# AI Skills Lab 1: Professional Power User

It's often said that LLMs should can be thought of as overconfident and all too eager to please junior coworkers. They will confidently tell you things that are incorrect and / or tell you that the most prozaic of your ideas are brilliant breakthroughs. GitHub copilot should be treated with the same level of scepticism and a greater level of caution because it can actually _do_ stuff based on its incorrect "understanding" of the world and the problem at hand. That said, when used carefully it's a powerful tool that will multiply your productivity.

## Careful Use

In this workshop, we'll cover several things that will help you to prevent GitHub Copilot from wreaking havoc on your codebase. Topics include...

- Guiding principles
- Ask, Edit and Agent chat modes
- Adding context
- Using `copilot-instructions.md`

## Theory into Practice

After a short demo, you'll be given the chance to practice careful use of GitHub copilot on a small refactoring exercise. In later workshops, you will add tests and then add new features to this same codebase.

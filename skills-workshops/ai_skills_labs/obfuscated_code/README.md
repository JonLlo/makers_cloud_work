# Obfuscated Code Refactoring Exercise

This repository contains intentionally obfuscated code samples in Java, JavaScript, and Python. The purpose is to provide material for refactoring exercises, helping developers practice improving code readability, maintainability, and structure.

## Structure

- `java/Mystery.java` — Obfuscated Java code sample
- `javascript/mystery.js` — Obfuscated JavaScript code sample
- `python/mystery.py` — Obfuscated Python code sample

## Running the Samples

### Java

To run the Java sample:

```sh
cd java
java Mystery.java
```

### JavaScript

To run the JavaScript sample:

```sh
cd javascript
node mystery.js
```

### Python

To run the Python sample:

```sh
cd python
python mystery.py
```

## Purpose

These files are intentionally hard to read. Use them for exercises in:
- Refactoring
- Improving code clarity
- Applying best practices

Feel free to fork or clone this repository and submit your own refactored versions!
